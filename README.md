# Backend API Geliştirme

## 1. Proje Yapısı Oluşturma
- Node.js ve Express.js kurulumu
- Klasör yapısı oluşturma (controllers, routes, data)
- package.json dosyası oluşturma ve gerekli paketleri yükleme

## 2. Veri Saklama
- JSON dosyaları kullanarak basit veri saklama
- posts.json ve users.json dosyaları oluşturma
- Dosya okuma/yazma işlemleri için fs modülü kullanımı

## 3. API Endpoint'leri Oluşturma
- Kullanıcı işlemleri (kayıt, giriş)
- Blog yazıları (ekleme, listeleme, güncelleme, silme)

## 4. Basit Kimlik Doğrulama
- JWT (JSON Web Token) entegrasyonu
- Kullanıcı yetki kontrolü middleware'i

## 5. Validasyon ve Hata Yönetimi
- Girdi doğrulama (validation) işlemleri
- Hata yakalama middleware'i
- Anlamlı hata mesajları döndürme

## 6. Temel Güvenlik Önlemleri
- CORS yapılandırması
- Helmet paketi ile temel güvenlik başlıkları
